### Method A: Google Search (Easiest)
1. Google your business name
2. Click on your Google Business Profile
3. Click "Ask for reviews"
4. Copy the link provided

### Method B: Google Business Profile Dashboard
1. Go to business.google.com
2. Click your business
3. Go to "Home" â†’ "Get more reviews"
4. Copy the short link

### Method C: Create Custom Link
Your Google review link format:
```

```

Find your Place ID: https://developers.google.com/maps/documentation/places/web-service/place-id

### Free QR Code Generators:

1. **QR Code Generator** (qr-code-generator.com)
   - Free, no signup
   - Custom colors available

2. **Canva** (canva.com)
   - Free with Canva account
   - Design around QR code

3. **QRCode Monkey** (qrcode-monkey.com)
   - Free, high resolution
   - Logo in center option

### Steps:
1. Go to generator site
2. Select "URL" type
3. Paste your Google review link
4. Customize colors to match your brand (optional)
5. Download as PNG (high resolution)

## âœ… QR CODE BEST PRACTICES

### DO:
- âœ… Make QR code at least 1" x 1" (bigger = easier to scan)
- âœ… Test on multiple phones before printing
- âœ… Use high contrast (dark code on light background)
- âœ… Add a clear call-to-action ("Scan to review")
- âœ… Place at eye level or easy reach

### DON'T:
- âŒ Make QR code too small
- âŒ Use low contrast colors
- âŒ Place where it's hard to reach
- âŒ Print on glossy surfaces (causes glare)
- âŒ Forget to test before mass printing

## ğŸ“Š TRACKING QR CODE SCANS

### Option 1: URL Shortener with Analytics
1. Shorten your Google review link with Bitly
2. Create QR code from the Bitly link
3. Check Bitly dashboard for scan stats

### Option 2: Google Analytics UTM
Add UTM parameters to track source:
```
[your-review-link]?utm_source=qrcode&utm_medium=tablecard
```

### Option 3: Multiple QR Codes
Create different QR codes for different locations:
- QR Code A = Table tents
- QR Code B = Leave-behind cards
- QR Code C = Receipts

Track which placement gets most reviews!
